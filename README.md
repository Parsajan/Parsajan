## 📂 Featured Projects

### ⚙️ [Advanced PID Controller — FPGA (Verilog)](https://github.com/Parsajan/fpga-pid-controller)

> Closed-loop industrial PID controller on Basys3 (Artix-7)

- **Q16.16 fixed-point arithmetic** — no floating-point unit required
- **Anti-windup** with conditional integration and output saturation
- **IIR derivative filter** to suppress high-frequency noise
- **SPI ADC interface** (MCP3201-compatible, 12-bit)
- **Runtime Kp/Ki/Kd tuning** via UART (115200 baud)
- **12-bit PWM output** (~12.2 kHz carrier)

`Verilog` `SPI` `UART` `PWM` `Fixed-Point` `FSM` `Vivado`

---

### 🌡️ [FPGA Sensor Hub — BME280 over SPI (VHDL)](https://github.com/Parsajan/fpga-sensor-hub)

> Multi-module sensor acquisition system on Basys3 (Artix-7)

- **BME280** temperature & humidity sensor via **SPI Mode 0**
- **Dual-FSM architecture**: independent acquisition and UART formatting FSMs
- **ASCII streaming** over UART — plug into any serial terminal
- **PWM LED** brightness proportional to measured temperature
- **Structural top-level** in VHDL — clean component-based design

`VHDL` `SPI` `UART` `BME280` `FSM` `Structural Architecture` `Vivado`
